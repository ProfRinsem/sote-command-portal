Purpose

Rocket’s Engineering Bay is where hardware can be staged before it enters the rack. It supports safer commissioning, recovery, and experimental work.

Student verbs

Stage, test, recover, prototype

Character layer

Professor Rocket

Current state

The current concept includes a workstation, monitor/keyboard setup, crash cart support, and lockable drawer/workbench storage for Rocket’s lab.

What students do here

Student activity

  • Prepare a server for rack deployment
  • Use a crash cart or bench workstation
  • Document hardware findings
  • Separate experimental work from stable services
Mission connections

Demo and future missions

  • Crash cart setup
  • Server prep station
  • Engineering change proposal
