# SOTE Portal Photo Drop Zone

Place approved, public-safe SOTE photos in this folder using the target names from `media-asset-checklist.html`.

Examples:

- `ceit-325-classroom-primary.jpg`
- `mission-control-primary.jpg`
- `rockets-forge-primary.jpg`
- `patch-bay-alpha-primary.jpg`
- `cyber-defense-command-primary.jpg`
- `student-action-first-signal.jpg`

Do not include passwords, credentials, private student information, sensitive IP addresses, unreviewed screenshots, or photos that expose sensitive labels.
