# SOTE Command Portal Content Guide

## Project board updates

Each project card should include:

- Project name
- Student-facing name
- Status
- Connected zone
- Connected character
- Connected pathway
- Current progress
- Next action
- Related mission or badge

## Status language

Use these status values consistently:

- Concept
- Proposal Drafted
- Ready to Build
- Build in Progress
- Operational / Online
- Blocked
- Future / Backlog

## Public safety rules

Do not publish passwords, production credentials, private student information, sensitive security findings, internal-only IP details, or unapproved diagrams.
